Below is a list of software packages we will be supporting in EasyBuild. Currently, EasyBuild is being cleaned up for public release; the EasyBuild framework has already been released, and we are now porting support for software packages implemented in _easyblocks_ to the public version of EasyBuild.

For each software package, we indicate in which (planned) version of EasyBuild support was/will be added.

The ones marked with (*) are a little bit higher on our TODO list for porting because they are recognized as commonly used software packages in High Performance Computing (HPC).
